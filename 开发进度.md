## 🎉 核心功能完成！

**选股策略系统核心模块已全部实现！**

### 快速开始

```bash
# 1. 下载数据（免费版）
cd examples/alpha_research
python download_data_simple.py

# 2. 检查数据
python check_data_quality.py

# 3. 测试股票池
python example_stock_pool.py

# 4. 测试财务数据
python example_fundamental_data.py

# 5. 运行综合测试
python test_stock_screener.py

# 6. 运行回测（需要真实数据）
python -c "
from datetime import datetime
from vnpy.alpha.lab import AlphaLab
from vnpy.alpha.strategy import ValueStockStrategy
from vnpy.alpha.strategy.cross_sectional_engine import create_cross_sectional_engine
from vnpy.trader.constant import Interval

lab = AlphaLab('./lab/test_strategy')
engine = create_cross_sectional_engine(lab, initial_capital=1_000_000)

engine.set_parameters(
    vt_symbols=['000001.SZ', '600036.SH', '600519.SH'],
    interval=Interval.DAILY,
    start=datetime(2024, 1, 1),
    end=datetime(2024, 12, 31)
)

engine.add_strategy(
    ValueStockStrategy,
    setting={'max_pe': 20, 'max_pb': 3, 'min_dividend_yield': 2, 'min_roe': 10}
)

engine.load_data()
engine.run_backtesting()
stats = engine.calculate_statistics()
print(f'总收益：{stats[\"total_return_pct\"]:.2f}%')
engine.show_chart()
"
```

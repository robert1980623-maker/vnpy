## 🎓 学习要点

### 1. 选股策略逻辑

**价值股策略条件**:
- PE < 20
- ROE > 10%
- 股息率 > 2%

**筛选过程**:
1. 从股票池获取股票列表
2. 读取财务数据 (PE, ROE, 股息率)
3. 应用筛选条件
4. 生成目标持仓列表

### 2. 交易信号生成

**买入信号**: 目标持仓中有，当前持仓中没有
**卖出信号**: 当前持仓中有，目标持仓中没有

**示例**:
```
当前持仓：[]
目标持仓：['000001.SZ', '600036.SH']
→ 买入：['000001.SZ', '600036.SH']
→ 卖出：[]
```

### 3. 仓位计算

**公式**:
```
单只股票金额 = 总资金 × 仓位比例
股票数量 = 单只金额 ÷ 价格 (取 100 的整数倍)
总成本 = 数量 × 价格 + 手续费
```

**示例**:
```
总资金：¥1,000,000
仓位比例：3%
单只金额：¥30,000

000001.SZ 价格：¥7.80
数量：30,000 ÷ 7.80 ≈ 3,800 股 → 3,800 股 (100 倍数)
实际金额：3,800 × 7.80 = ¥29,640
```

### 4. 费用计算

**手续费**: 成交金额 × 0.0003 (万分之三)
**印花税**: 卖出时收取，成交金额 × 0.001 (千分之一)
**滑点**: 价格 × 0.01 (1%)

**示例**:
```
买入 000001.SZ 10,000 股 @ ¥7.80
成交金额：78,000
手续费：78,000 × 0.0003 = 23.40
滑点：7.80 × 0.01 = 0.078
实际价格：7.80 + 0.078 = 7.878
总成本：78,000 + 23.40 = 78,023.40
```

## 💡 心得体会

### 优点

1. ✅ **流程清晰**: 选股 → 信号 → 下单 → 持仓，逻辑清楚
2. ✅ **参数灵活**: 可通过配置文件调整策略参数
3. ✅ **数据完整**: 保存所有交易记录和持仓信息
4. ✅ **风险可控**: 模拟交易无真实资金风险

### 注意事项

1. ⚠️ **数据质量**: 依赖历史数据的准确性
2. ⚠️ **滑点设置**: 实盘滑点可能更大
3. ⚠️ **流动性**: 未考虑实际流动性限制
4. ⚠️ **市场冲击**: 大资金可能影响价格

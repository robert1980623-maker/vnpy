## 📈 性能分析

### 理论性能

**优化前（串行）**:
```
5 只股票 × 3 秒/只 = 15 秒
+ 延迟：4 × 0.5 秒 = 2 秒
总计：~17 秒
```

**优化后（并行，5 线程）**:
```
5 只股票 ÷ 5 线程 = 1 批
1 批 × 3 秒/批 = 3 秒
+ 延迟：0 秒
总计：~3 秒（理论）
```

**实际性能**:
```
总耗时：18.3 秒
原因：网络问题导致重试
```

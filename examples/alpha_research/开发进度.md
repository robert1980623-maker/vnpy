**使用示例**:

```bash

# 自定义资金和买入数量

| `--capital` | 初始资金 | 1,000,000 |
| `--volume` | 每只股票买入数量 | 1000 |
| `--log-level` | 日志级别 (DEBUG/INFO/WARNING/ERROR) | INFO |

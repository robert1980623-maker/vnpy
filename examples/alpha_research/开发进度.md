### 1. 创建主入口脚本 `main.py` ⭐⭐⭐⭐⭐

**文件**: `examples/alpha_research/main.py`

**功能**:
- ✅ 一键运行完整流程（数据下载 → 选股 → 回测 → 模拟交易）
- ✅ 灵活的命令行参数
- ✅ 日志系统集成
- ✅ 简化模式支持（依赖缺失时仍可运行）
- ✅ 错误处理和异常恢复

**使用示例**:

```bash
# 完整流程
python3 main.py --strategy value --stocks 10

# 仅选股和模拟交易（跳过下载和回测）
python3 main.py --strategy value --stocks 10 --skip-download --skip-backtest

# 仅下载数据
python3 main.py --only-download --max 20

# 自定义资金和买入数量
python3 main.py --strategy momentum --stocks 5 --capital 2000000 --volume 2000

# 查看帮助
python3 main.py --help
```

**命令行参数**:

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--strategy, -s` | 选股策略 (value/growth/momentum/quality/industry) | value |
| `--stocks, -n` | 选股数量 | 10 |
| `--skip-download` | 跳过数据下载 | False |
| `--skip-backtest` | 跳过回测 | False |
| `--skip-trading` | 跳过模拟交易 | False |
| `--only-download` | 仅下载数据 | False |
| `--capital` | 初始资金 | 1,000,000 |
| `--volume` | 每只股票买入数量 | 1000 |
| `--log-level` | 日志级别 (DEBUG/INFO/WARNING/ERROR) | INFO |
| `--log-dir` | 日志目录 | ./logs |

## 价值投资策略

### 核心逻辑

**买入被市场低估的股票，等待价值回归**

### 关键指标

| 指标 | 公式 | 说明 | 合理范围 |
|------|------|------|----------|
| PE (市盈率) | 股价/每股收益 | 估值水平 | <20 |
| PB (市净率) | 股价/每股净资产 | 资产估值 | <3 |
| PS (市销率) | 市值/营业收入 | 销售估值 | <5 |
| PEG | PE/净利润增长率 | 成长估值 | <1 |
| 股息率 | 每股分红/股价 | 分红回报 | >2% |

### 选股条件

```python
价值股筛选条件:
├── PE < 20 (低于行业平均)
├── PB < 3 (资产被低估)
├── ROE > 10% (盈利能力良好)
├── 股息率 > 2% (有分红)
└── 负债率 < 60% (财务安全)
```

### 优点与风险

| 优点 | 风险 |
|------|------|
| 安全边际高 | 价值陷阱 |
| 下行风险有限 | 回归时间不确定 |
| 适合长期持有 | 可能错过成长股 |

### 经典案例

- **巴菲特**: 可口可乐、美国运通
- **格雷厄姆**: 烟蒂股策略

## 成长投资策略

### 核心逻辑

**买入高增长公司，分享成长红利**

### 关键指标

| 指标 | 说明 | 优秀标准 |
|------|------|----------|
| 营收增长率 | 营业收入同比增长 | >25% |
| 净利润增长率 | 净利润同比增长 | >30% |
| ROE | 净资产收益率 | >15% |
| 毛利率 | 毛利润/营收 | >30% |
| 研发占比 | 研发费用/营收 | >5% |

### 选股条件

```python
成长股筛选条件:
├── 营收增长率 > 25% (连续 3 年)
├── 净利润增长率 > 30% (连续 3 年)
├── ROE > 15% (持续盈利)
├── 毛利率 > 30% (竞争优势)
└── 行业景气度高 (赛道好)
```

### 优点与风险

| 优点 | 风险 |
|------|------|
| 收益潜力大 | 估值波动大 |
| 复利效应强 | 成长可能失速 |
| 适合长期持有 | 需要持续跟踪 |

### 成长阶段

```
公司成长周期:
├── 初创期 (高风险高回报)
├── 成长期 (最佳投资窗口)
├── 成熟期 (稳定收益)
└── 衰退期 (避免投资)
```

## 动量策略

### 核心逻辑

**追涨杀跌，顺势而为**

### 动量指标

| 指标 | 周期 | 说明 |
|------|------|------|
| 12 月动量 | 12 个月收益率 | 长期动量 |
| 6 月动量 | 6 个月收益率 | 中期动量 |
| 1 月动量 | 1 个月收益率 | 短期动量 |
| RSI | 14 日 | 相对强弱 |
| MACD | 12/26/9 | 趋势指标 |

### 选股条件

```python
动量股筛选条件:
├── 12 月收益率排名前 20%
├── 股价在 250 日均线上方
├── 250 日均线向上
├── 成交量放大
└── 相对强度 RS > 80
```

### 交易规则

```
买入信号:
├── 突破 52 周新高
├── 均线多头排列
└── 成交量确认

卖出信号:
├── 跌破 20 日均线
├── 动量反转
└── 止损 8-10%
```

### 优点与风险

| 优点 | 风险 |
|------|------|
| 趋势明确 | 震荡市失效 |
| 收益可观 | 回撤较大 |
| 规则清晰 | 需要严格止损 |

## 质量策略

### 核心逻辑

**买入高质量公司，长期持有**

### 质量指标

| 指标类别 | 具体指标 | 优秀标准 |
|----------|----------|----------|
| 盈利能力 | ROE | >15% |
| 盈利能力 | ROA | >8% |
| 盈利能力 | 毛利率 | >30% |
| 财务健康 | 资产负债率 | <50% |
| 财务健康 | 流动比率 | >1.5 |
| 财务健康 | 利息保障倍数 | >5 |
| 经营效率 | 资产周转率 | 行业前 30% |
| 经营效率 | 存货周转率 | 行业前 30% |

### 选股条件

```python
质量股筛选条件:
├── ROE > 15% (连续 5 年)
├── 毛利率 > 30% (稳定)
├── 资产负债率 < 50%
├── 经营现金流为正
└── 无重大财务风险
```

### 质量因子评分

```python
质量评分 = 0.3 * ROE 得分 + 0.2 * 毛利率得分 + 
          0.2 * 财务健康得分 + 0.15 * 经营效率得分 +
          0.15 * 盈利稳定性得分
```

### 优点与风险

| 优点 | 风险 |
|------|------|
| 公司质地好 | 估值可能偏高 |
| 抗风险能力强 | 短期可能跑输 |
| 适合长期持有 | 需要耐心 |

## 高股息策略

### 核心逻辑

**买入高分红股票，获取稳定现金流**

### 关键指标

| 指标 | 公式 | 说明 | 合理范围 |
|------|------|------|----------|
| 股息率 | 每股分红/股价 | 分红回报 | >3% |
| 分红率 | 分红/净利润 | 分红意愿 | 30-70% |
| 股息增长率 | 股息年增长 | 增长能力 | >5% |
| 连续分红年数 | - | 稳定性 | >5 年 |

### 选股条件

```python
高股息股筛选条件:
├── 股息率 > 3% (高于债券)
├── 分红率 30-70% (可持续)
├── 连续分红 > 5 年 (稳定)
├── 盈利稳定 (无大幅波动)
└── 现金流充足 (能支撑分红)
```

### 适合人群

- ✅ 退休人士 (需要现金流)
- ✅ 保守投资者 (追求稳定)
- ✅ 长期投资者 (复利效应)
- ✅ 熊市防御 (抗跌性强)

### 优点与风险

| 优点 | 风险 |
|------|------|
| 稳定现金流 | 成长性有限 |
| 抗跌性强 | 利率敏感 |
| 复利效应 | 可能错过成长股 |

## 多因子策略

### 核心逻辑

**综合多个因子，全面评估股票**

### 因子体系

```
因子库:
├── 价值因子 (30% 权重)
│   ├── PE
│   ├── PB
│   └── PS
├── 成长因子 (25% 权重)
│   ├── 营收增长
│   ├── 利润增长
│   └── PEG
├── 质量因子 (25% 权重)
│   ├── ROE
│   ├── 毛利率
│   └── 负债率
├── 动量因子 (10% 权重)
│   ├── 12 月收益
│   └── RSI
└── 情绪因子 (10% 权重)
    ├── 成交量变化
    └── 资金流向
```

### 评分模型

```python
综合得分 = Σ(因子值 × 因子权重)

步骤:
1. 数据标准化 (Z-Score)
2. 行业中性化
3. 加权求和
4. 排序选股
```

### 选股流程

```
1. 初筛 (剔除 ST、次新、流动性差)
   ↓
2. 因子计算 (计算各因子值)
   ↓
3. 标准化处理 (Z-Score)
   ↓
4. 加权评分 (综合得分)
   ↓
5. 排序选股 (选前 N 只)
   ↓
6. 仓位配置 (等权或优化)
```

### 优点与风险

| 优点 | 风险 |
|------|------|
| 全面评估 | 因子可能失效 |
| 分散风险 | 模型复杂 |
| 可解释性强 | 需要持续优化 |

## 行业轮动策略

### 核心逻辑

**根据经济周期，轮动配置不同行业**

### 美林时钟

```
经济周期与行业配置:

复苏期 (低增长低通胀)
├── 配置：股票
└── 行业：科技、金融

过热期 (高增长高通胀)
├── 配置：商品
└── 行业：资源、材料

滞胀期 (低增长高通胀)
├── 配置：现金
└── 行业：防御、公用事业

衰退期 (低增长低通胀)
├── 配置：债券
└── 行业：必需消费、医疗
```

### 行业分类

```
周期性行业:
├── 金融 (银行、保险、券商)
├── 房地产
├── 原材料 (钢铁、煤炭)
├── 工业 (机械、建筑)
└── 能源 (石油、天然气)

防御性行业:
├── 必需消费 (食品、饮料)
├── 医疗健康
├── 公用事业 (电力、水务)
└── 电信服务

成长性行业:
├── 科技 (软件、硬件)
├── 通信服务
└── 可选消费 (汽车、家电)
```

### 轮动信号

```
判断指标:
├── GDP 增长率
├── CPI/PPI
├── PMI
├── 利率水平
├── 货币供应量
└── 行业景气指数
```

### 优点与风险

| 优点 | 风险 |
|------|------|
| 把握周期 | 周期判断困难 |
| 超额收益 | 轮动时机难把握 |
| 分散风险 | 交易成本高 |

## 策略组合建议

### 保守型组合

```
目标：年化 10-15%, 最大回撤<15%

配置:
├── 价值策略 40%
├── 高股息策略 30%
├── 质量策略 20%
└── 现金 10%

特点:
- 稳健为主
- 注重安全边际
- 适合风险厌恶者
```

### 平衡型组合

```
目标：年化 15-25%, 最大回撤<25%

配置:
├── 多因子策略 40%
├── 成长策略 25%
├── 价值策略 20%
├── 动量策略 10%
└── 现金 5%

特点:
- 均衡配置
- 攻守兼备
- 适合大多数投资者
```

### 激进型组合

```
目标：年化 25%+, 最大回撤<35%

配置:
├── 成长策略 35%
├── 动量策略 30%
├── 多因子策略 25%
├── 行业轮动 10%
└── 现金 0%

特点:
- 追求高收益
- 承受高波动
- 适合风险承受力强者
```

## 📝 总结

### 策略选择原则

1. **了解自己**: 风险偏好、投资目标、时间周期
2. **分散配置**: 不要把所有鸡蛋放在一个篮子里
3. **持续学习**: 市场在变，策略需要进化
4. **严格执行**: 纪律是成功的关键
5. **风险控制**: 活着最重要

### 策略有效性

```
策略生命周期:
├── 发现期 (少数人知道，效果好)
├── 成长期 (越来越多人用，效果尚可)
├── 成熟期 (广泛使用，效果一般)
└── 衰退期 (过度使用，效果差)

应对:
- 持续研发新策略
- 多策略组合
- 动态调整权重
```

### 成功公式

```
成功 = 正确的方法 × 严格的执行 × 时间的复利
```
